> Nutro components - we're constantly updating more components daily! <br />
> Explore the npm package [here](https://www.npmjs.com/package/nutro-components)

The best way to use our components is

#### Using NPM:

```bash
npm i nutro-components
```

or

#### Using Yarn:

```bash
yarn add nutro-components
```

If you find any bugs/feedbacks, feel free to write to us at mariappangameo@gmail.com

Once you have installed the dependency, you can explore our components doc & it's usage. Happy plug & play with nutro components!
